It features easy management of bots, chat history, and presets + more.

Total features (as of 4/30/26):

- Lorebooks (creation, usage, editing)
- Character browsing (from 10+ sources)
- Character editing, creation and importing
- Persona management (creation, usage, editing)
- Preset management (importing, toggling)
- Preset browser (quick imports)
- Progressive Disclosure style design (avoid being too simplistic or too complex)
- Customization options (gif/img backgrounds, color pallet augmentations)

## Installation
Download the git repo (https://github.com/kiwiaichat/kiwiai) and open the index.html file in your browser.

Alternatively, you can use https://kiwiaichat.github.io/kiwiai/ to access the hosted version (static via GitHub).

## Self Hosting

### IOS

On IOS, it is reccomended that you utilize the webclip, for easy access to the app from your home screen. You can install it by clicking on [this link](https://kiwiaichat.github.io/kiwiai/KiwiAI.mobileconfig). To install this config, head into settings, you'll have a notification, press it, install the webclip (essensially just a website, but in app form), and you're good to go.

### Android

Currently, there is no "home screen" style installation for android, but you can still use https://kiwiaichat.github.io/kiwiai/ via your browser and add it to your homescreen manually.
